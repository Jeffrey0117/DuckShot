## ✨ 功能特色

- 🎯 **三種截圖模式** — 區域截圖、全螢幕截圖、視窗截圖
- 🖼️ **高畫質擷取** — 依顯示器縮放比例（HiDPI / DPR 感知）以實體像素抓圖，文字銳利不糊
- ⚡ **極速啟動** — 按下快捷鍵約 0.1 秒覆蓋層即出現
- 🔤 **文字辨識（OCR）** — 框選文字按 `5`，視窗文字直讀 → PaddleOCR → Tesseract 三層引擎，可選用 Gemini AI 重新辨識
- 🐤 **截圖自動隱藏滑鼠游標** — 畫面乾淨不留游標，並以鴨鴨指標 + 對位點精準對齊
- 🗂️ **存檔後快速分類** — 存檔即跳通知，一鍵（或數字鍵）把截圖移到分類資料夾
- 📁 **可瀏覽圖庫** — 圖庫分頁 = 真實資料夾，截圖歷史一目了然，可刪除（移至資源回收桶）
- 📋 **快速複製 / 儲存** — 一鍵複製到剪貼簿、儲存時自動複製、儲存位置可自訂（預設：圖片 / DuckShot）
- ☁️ **上傳分享** — 一鍵上傳圖床（URUSAI!）取得分享連結
- ⌨️ **全域快捷鍵** — 不必切回程式，隨時開抓（可自訂）
- 📌 **置頂 / 系統托盤** — 視窗可置頂，可最小化到系統托盤常駐

## ⌨️ 預設快捷鍵

| 動作 | 快捷鍵 |
|------|--------|
| 區域截圖 | `Alt + A` |
| 全螢幕截圖 | `PrintScreen` |
| 視窗截圖 | `Alt + W` |

> 皆可在設定頁自訂。

**截圖覆蓋層內：** `Enter` / `2` 儲存、`1` / `Ctrl+C` 複製、`3` 上傳、`4` 連續截圖、`5` 文字辨識、右鍵取消選取（無選取時再按一次關閉）、`Esc` 取消。

## 📥 安裝方式

1. 點上方按鈕下載 **`DuckShot-Setup-1.1.1.exe`**
2. 執行安裝檔（可自選安裝路徑，會建立桌面與開始選單捷徑）
3. 開啟「DuckShot」即可開始截圖 🦆

> Windows 10 / 11（64 位元）。安裝檔未經數位簽章，首次執行時 Windows SmartScreen 可能提示，點「**其他資訊 → 仍要執行**」即可。

## 💻 系統需求

- Windows 10 或 Windows 11（x64）
